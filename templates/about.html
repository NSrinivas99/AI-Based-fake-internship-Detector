ℹ️ About Fake Internship Detector

System details, machine learning pipeline, heuristics engine, and final year project documentation.

🛡️ Project Objective

College students and freshers frequently fall victim to recruitment scams on chat applications like WhatsApp and Telegram, or through fraudulent clone websites. These scams often request "security deposits", "registration fees", or attempt to harvest private personal data.

This system provides an integrated defense mechanism to screen job offers through four separate vectors: Natural Language Processing, suspicious keywords analysis, company legitimacy checks, and domain SSL scans.

🤖 NLP and Machine Learning Architecture

The classifier is trained on a synthetic dataset containing 1,000 postings (500 genuine and 500 fake/scam offers). The pipeline runs as follows:

  • Preprocessing: Text strings are lowercased, punctuation and special symbols are removed, and the remaining tokens are filtered against standard NLTK stopwords.
  • TF-IDF Vectorization: Converts clean text tokens into numerical vectors using term frequency-inverse document frequency weighting with an ngram range of (1,2) to capture word pairings.
  • Classifiers: The system trains both Logistic Regression and Random Forest models, picking the one with the highest F1-score as the active defense model.

📊 Risk Weighting & Scoring Engine

The system calculates a composite risk index (0 to 100) using a multi-factor weighted equation:

ANALYSIS FACTOR WEIGHT
Machine Learning Model Prediction 40%
Keyword Analysis 25%
Stipend Realism 15%
Company Legitimacy 10%
URL Safety 10%
